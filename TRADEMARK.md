# The Privex name

The Privex source code is licensed under **AGPL-3.0-or-later** (see [LICENSE](LICENSE)).
That license covers the code. It does not grant rights to the **Privex** name, logo,
or visual identity.

You are welcome — encouraged — to fork the code, read it, run it, and build on it
under the AGPL. If you ship a modified version, please give it your own name. Don't
call it "Privex," and don't use the Privex name or branding in a way that implies your
build is the official project or is endorsed by it.

This is the same line most trusted open projects draw (Signal, Mozilla, and others do
the same): the code is free for anyone to use and improve, but the *name* points at one
specific, accountable thing, so a user who installs "Privex" always knows exactly what
they are trusting with their safety.

For permission to use the name, or any question about this, open an issue or reach the
maintainer through the links in the [README](README.md).
